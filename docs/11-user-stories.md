# 11 - User Stories

## What is a User Story?
A user story is a short, simple description of a feature told from the perspective of the user. Format: **"As a [role], I want to [action], so that [benefit]."**

---

## Customer Stories

| ID  | User Story                                                                                       | Priority |
|-----|--------------------------------------------------------------------------------------------------|----------|
| US-01 | As a customer, I want to register an account, so that I can make and manage bookings.          | High     |
| US-02 | As a customer, I want to log in securely, so that my bookings and data are private.            | High     |
| US-03 | As a customer, I want to view all available services, so that I can choose what I need.         | High     |
| US-04 | As a customer, I want to see available time slots for a service, so that I can pick a time that suits me. | High |
| US-05 | As a customer, I want to book an appointment, so that I can reserve my preferred time slot.    | High     |
| US-06 | As a customer, I want to receive an email confirmation, so that I know my booking was successful. | High  |
| US-07 | As a customer, I want to view my upcoming bookings, so that I remember when my appointments are.| High    |
| US-08 | As a customer, I want to cancel a booking, so that I can free up the slot if I can't attend.   | Medium   |
| US-09 | As a customer, I want to view my past bookings, so that I can track my history.                | Low      |

## Admin Stories

| ID  | User Story                                                                                       | Priority |
|-----|--------------------------------------------------------------------------------------------------|----------|
| US-10 | As an admin, I want to log in to a separate dashboard, so that I can manage the business.     | High     |
| US-11 | As an admin, I want to add services with name, duration, and price, so that customers can book them. | High |
| US-12 | As an admin, I want to edit or delete services, so that I can keep the listing up to date.     | High     |
| US-13 | As an admin, I want to define available time slots/working hours, so that bookings only happen at valid times. | High |
| US-14 | As an admin, I want to view all upcoming bookings, so that I can plan my day.                  | High     |
| US-15 | As an admin, I want to cancel any booking, so that I can handle cancellations from my end.     | Medium   |
| US-16 | As an admin, I want to see a daily schedule view, so that I can quickly understand my workload.| Medium   |
| US-17 | As an admin, I want to view all registered customers, so that I can reference past clients.    | Low      |
