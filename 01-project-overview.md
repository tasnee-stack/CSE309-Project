# 01 - Project Overview

## Project Name
**BookEase** – An Online Appointment & Reservation Booking System

## Project Summary
BookEase is a web-based booking and reservation system that allows users to schedule appointments or reserve services online. The platform targets small to medium-sized businesses (such as salons, clinics, tutoring centers, and fitness studios) that need a simple, reliable way to manage bookings without relying on phone calls or manual scheduling.

## Project Goals
- Allow customers to view available time slots and book appointments online
- Allow business owners/admins to manage their schedule, services, and reservations
- Send booking confirmations and reminders to users
- Reduce scheduling conflicts and no-shows through an automated system

## Technology Stack
| Layer       | Technology         |
|-------------|--------------------|
| Frontend    | React.js            |
| Backend     | Node.js + Express.js |
| Database    | MongoDB (Mongoose) |
| Auth        | JWT (JSON Web Tokens) |
| Styling     | Tailwind CSS        |
| Deployment  | Vercel (Frontend) + Railway/Render (Backend) |

## Project Scope
**In Scope:**
- User registration and login (Customer & Admin roles)
- Service listing and management
- Calendar-based appointment booking
- Admin dashboard to view, confirm, and cancel bookings
- Email notifications (confirmation on booking)

**Out of Scope (for this version):**
- Payment processing
- Mobile native apps
- Multi-location business support
